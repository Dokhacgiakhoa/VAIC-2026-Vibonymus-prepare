👥 Phân công vai trò (gợi ý — chỉnh theo thế mạnh thật của team)

KAI
Kỹ thuật & AI
  • Kiến trúc hệ thống, lựa chọn stack
  • Tích hợp AI/model, xử lý logic lõi
  • Backend, API, deploy live URL
  • Đảm bảo sản phẩm chạy ổn định lúc demo
Quân
Product & Data
  • Phân tích đề bài, xác định user & bài toán
  • Cấu trúc dữ liệu, luồng sản phẩm
  • Hỗ trợ frontend / test luồng end-to-end
  • Ghi AI collaboration log xuyên suốt
Hiếu
Design, UX & Pitch
  • UI, prototype, trải nghiệm người dùng
  • Chuẩn bị slide, kịch bản pitch
  • Quay & dựng demo video (≤5 phút)
  • Người trình bày chính nếu vào Top 10
Vai trò chỉ là trọng tâm — ở các mốc quan trọng (chọn track, 2 checkpoint, đêm cuối) cả 3 người cùng vào việc.

Ideation & tài liệu
Claude / ChatGPT
Brainstorm hướng giải quyết đề bài, viết project description, outline pitch.
Coding assistant
Claude Code / Cursor / Copilot
Sinh code, refactor, debug nhanh trong 48h — KAI chủ trì, cả team dùng để tăng tốc.
UI / Prototyping
v0, Lovable, Figma AI
Dựng giao diện nhanh, tạo prototype để Hiếu tinh chỉnh UX.
Model / Data (giải phụ)
PyTorch
Nếu bài toán cần train/fine-tune model — có cơ hội tranh Best PyTorch Award ($5,000).
